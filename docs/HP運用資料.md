## 4. HP 情報設計の決定事項

- セクション順: ヒーロー → 悩み → こんな方 → できること → 注目の商品 → ラインナップ → 記事
- 「悩み」は工房レベルの悩み（工事台帳固有ではない）
- 「できること」= 2本柱（道具＋情報）。業務テンプレートカードは除外戦略のため削除
- お問い合わせCTAは降格。主ナビは「ツール・商品を見る」と FAQ
- ヒーローのチェックバッジ = サイト全体価値（IT苦手でも使える／建設業の実務に特化）。商品固有の主張にしない
- 空セクション（お客様の声 準備中／第3弾 構想中）は意図的に残置。ただし建設番頭の名は出さない
- **除外戦略**: 無料テンプレート配布はしない／見積・請求・工程管理はライト版スコープ外／作業量増加を伴う施策は採らない（副業・受動運用前提）

## 5. ワークフローと教訓

- 役割分担: **Claude = 指示書生成 / Claude Code = 実装**
- 指示書には必ず **着手確認（§0）** を入れる。既存記事・コミットを先に点検し、着手済みなら無断上書きしない（二重作成防止。資格記事で実際に功を奏した）
- **comp 忠実再現の教訓**: 「既存の Astro 構造に従え」と書くと Claude Code は既存の簡略版を温存してしまう。comp HTML を「唯一の真実」とし、「既存実装は破棄して置換せよ（preserve するな）」と明示する（指示書#35 で確立）
- 記事追加時は、先に `config.ts` と既存記事を読ませて **frontmatter スキーマ・カテゴリ enum・命名規則を踏襲** させる
- **存在しない URL へリンクさせない**（商品ページ・関連記事は実在確認時のみリンク）
- **コンテンツ制作の分担（重要な教訓）**: Claude Code にコンテンツマップだけ渡して記事本文を自作させると、構造・トーン・SEO/AEO型は良好だが、**法令の条番号・罰則額・税率・制度区分など細部で事実誤りが出やすい**（実際にA群で自賠責の罰則額・酸欠健診・通達番号、B/C群で2割特例の期限・簡易課税の業種区分・雇用保険料率などの誤りが発生し、複数回の精査・修正を要した）。今後の記事追加は、**Claudeが事実確認済みの本文確定稿を書いてから Claude Code に渡し、Claude Code は実装に徹する**分担を基本とする。マップだけ渡して一気に自作させると、精査と修正の往復コストが大きくなる。
- **事実の鮮度**: 料率・制度・特例には有効期限や年度改定がある（雇用保険料率＝毎年度改定、労災保険率＝原則3年ごと、インボイス2割特例＝個人は令和8年分まで 等）。記事に年度・時点を明記し、「最新は公式（厚労省・国税庁・年金機構等）で確認」の注記を添える。

## 6. SEO 運用方針

- 副業・受動運用。月次で Search Console をチェック、確定申告期（1〜3月）に集中。能動的な力は製品開発へ回す
- GA4 の前期比%（アクティブユーザー・イベント数等）は母数が小さくノイズ。内部アクセスでも振れる。**当面は無視**。本質は Search Console の検索流入
- **追うべき指標はクリック（と、それが取れているページ）**。表示回数は単独では意味を持たない

## 7. アナリティクスの読み方（重要知見）

- **2024年問題ガイド記事**（`kensetsu-2024-mondai-guide`）は「国土交通省」「厚生労働省」「公式」を含む公式志向クエリで大量表示されるが、検索者が探しているのは公式ページなのでクリックされない。7日間で **表示5万・クリック0** を記録して確定した
- 教訓: **表示の爆発 ≠ 成果**。原因が「検索意図のズレ（＋順位が9〜10位と低い）」なので、title/description を直しても回収できない。**追わない・放置が正解**
- ただし「建設業2024/2026年問題に関連するサイト」という **ドメインのテーマ認識** は資産として残る（隣接テーマを出したとき拾われやすくなる下地）
- **「効く型」= 検索意図と噛み合う安全衛生・実務解説**。実証済み: 一人親方の安全衛生2025記事（`hitori-oyakata-anzeneisei-2025`、CTR 約5%）
- 通常の Google ウェブ検索結果にサムネイル画像は出ない。打ち手は title / description（スニペット）のみ。見出し改善が効くのは「意図が合っていて・そこそこ良い順位で・なのに CTR が伸びない」ページ（一人親方記事・products など）

## 8. コンテンツ戦略

- 戦略は1本＝「**検索意図と噛み合う実務解説を淡々と増やす**」。題材は2系統から取る:
  - **実証枠**: 安全衛生・資格・法令の実務解説（クリックが取れる型。優先）
  - **検証枠**: 2026年問題まわりで公式と競合しない、収支・管理側の隣接ネタ（工事台帳が自然に答えになる）
- **露出を増やすこと自体を目的にしない**（2024年問題ガイドの5万表示0クリックが反面教師）
- 高表示・低クリックの記事には、効く記事への **内部リンクで回遊の仕事** をさせる
- 進め方は片手間前提。カレンダーで縛らず、キューから1本ずつ指示書化して出す
