確定申告で帳簿をつける際、「これはどの科目に入れればいいのか」で迷う方は多いです。この記事では、**建設業・一人親方がよく使う勘定科目と仕訳の書き方**を一覧でまとめます。

## 売上（収入）の科目

| 科目 | 内容 | 例 |
|------|------|-----|
| 売上高 | 工事の請負代金 | ○○工事 完工 請負金額300万円の入金 |
| 雑収入 | 本業以外の臨時収入 | 廃材の売却代金、補助金受取 等 |

> 消費税の申告（課税事業者の場合）では、売上高は税抜き金額で記録します。

## 経費（必要経費）の主な科目

### 工事に直接かかる費用

| 科目 | 内容 | 具体例 |
|------|------|--------|
| 外注費 | 下請けへの支払い | 職人・業者への発注代金 |
| 材料費 | 工事に使用する資材 | 木材・コンクリート・配管材料 等 |
| 工具器具備品 | 工具・小型機器の購入 | ドリル・レーザー墨出し器（10万円未満） |
| 消耗品費 | 消耗する部材・用品 | 砥石・ビス・養生テープ 等 |

> 10万円以上の工具・機器は「固定資産」として減価償却が必要です（青色申告で30万円未満は特例で一括経費算入できます）。

### 車両・交通費

| 科目 | 内容 | 具体例 |
|------|------|--------|
| 車両費 | 仕事で使う車の維持費 | 車検費用・オイル交換・タイヤ代 |
| 燃料費 | ガソリン代 | 仕事での移動・現場への往復 |
| 旅費交通費 | 交通費・宿泊費 | 電車・バス代、遠方の現場での宿泊 |
| 高速道路料金 | ETCや料金所の利用 | 現場への移動・材料の仕入れ |

> プライベートでも使う車は、仕事で使う割合（按分）だけを経費にします。走行日誌をつけておくと按分の根拠になります。

### 保険・資格・教育

| 科目 | 内容 | 具体例 |
|------|------|--------|
| 損害保険料 | 工事保険・賠償保険 | 工事賠償責任保険の保険料 |
| 保険料 | 労災・組合費 | 一人親方労災特別加入の保険料・組合費 |
| 研修費 | 資格・講習費用 | 技能講習・特別教育の受講料、テキスト代 |

### 事務・通信費

| 科目 | 内容 | 具体例 |
|------|------|--------|
| 通信費 | 電話・インターネット | 仕事用スマートフォンの通信料 |
| 事務用品費 | 事務消耗品 | ボールペン・コピー用紙・印刷代 |
| 広告宣伝費 | 集客・宣伝 | 名刺印刷・チラシ・ウェブサービス利用料 |

### その他

| 科目 | 内容 | 具体例 |
|------|------|--------|
| 地代家賃 | 事務所・駐車場の賃料 | 現場事務所の賃料、資材置き場 |
| 雑費 | 他の科目に当てはまらない経費 | 現場のゴミ袋・清掃費 等（少額のもの） |
| 租税公課 | 税金・公課 | 事業用車両の自動車税・印紙代 |

## よくある仕訳例

**① 工事代金の入金**
```
（借方）普通預金 3,000,000  /  （貸方）売上高 3,000,000
```

**② 外注業者への支払い**
```
（借方）外注費 800,000  /  （貸方）普通預金 800,000
```

**③ 材料費の購入（現金払い）**
```
（借方）材料費 50,000  /  （貸方）現金 50,000
```

**④ 車の燃料費（按分60%）**
```
（借方）燃料費 6,000  /  （貸方）現金 10,000（ガソリン代10,000円のうち60%）
```

> 白色申告では複式簿記（借方・貸方）の記帳は不要です。収入・支出の記録だけで構いません。

## 科目の迷いを減らすコツ

**「何のために使ったか」で考える**：勘定科目は一度決めたら年間で統一することが重要です。同じ支出を年によって違う科目に入れると、年間比較ができなくなります。

**迷ったら「雑費」より別科目に**：「雑費」が多くなると税務調査で説明が難しくなります。できるだけ実態に合った科目を使うか、「消耗品費」「雑費」のいずれかに統一します。

## まとめ

建設業の確定申告でよく使う勘定科目は、売上・外注費・材料費・車両費・損害保険料が中心です。**科目を年間で統一し、領収書と対応させて記録することが基本**です。科目に迷う場合は、税理士への相談が確実です。
