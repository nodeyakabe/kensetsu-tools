「実行予算」という言葉を聞いたことはあるでしょうか。建設業では受注後に「この工事をいくらのコストで完成させるか」を計画する実行予算が、原価管理の出発点になります。この記事では、**実行予算とは何か・なぜ必要か**を基本から解説します。

## 実行予算とは

実行予算とは、受注した工事を完了するために**実際にかかるコストの計画**です。見積もりとは目的が異なります。

| | 見積もり | 実行予算 |
|---|------|------|
| 目的 | 発注者への提示（受注価格の根拠） | 社内での原価管理（利益を出すための計画） |
| タイミング | 受注前 | 受注後・着工前 |
| 視点 | 発注者が納得できる金額 | 自社がどう施工するかの計画 |
| 主な内容 | 材料・労務・諸経費の積算 | 外注費・材料費・労務費の実施計画 |

見積もりは「いくらで売るか」、実行予算は「いくらで作るか」です。

## 実行予算の主な構成項目

| 項目 | 内容 |
|------|------|
| 労務費 | 直接工事に従事する職人・作業員の賃金 |
| 外注費 | 下請け業者への発注金額 |
| 材料費 | 工事に使用する資材・建材の購入費 |
| 機械経費 | 重機・機材のリース費・燃料費 |
| 現場経費 | 仮設・廃棄物処理・安全費 等 |
| 諸経費 | 現場事務所費・交通費・通信費 等 |

これらを工事ごとに積み上げたものが実行予算の合計です。

## 実行予算がないと起きる問題

実行予算を作らずに工事を進めると、次のような問題が起きやすくなります。

**① 赤字に気づくのが遅れる**
工事が終わって請求書を並べてみると赤字だった、というケースがよくあります。実行予算があれば、途中で原価が計画を超えた時点で対策を打てます。

**② 外注費の交渉根拠がない**
「この工事にどれくらい払えるか」の上限が不明なまま、外注業者との交渉になります。結果として利益を削る発注になりがちです。

**③ 会社全体の資金繰りが読めない**
工事ごとの支払いスケジュールが不明だと、いつ・いくらの資金が必要かを予測できません。

## 実行予算の作り方（基本ステップ）

**① 請負金額から目標粗利を決める**
まず「この工事でいくら粗利を残したいか」を設定します。

例：請負金額500万円 → 目標粗利20%（100万円）→ 原価上限400万円

**② 原価上限を各費目に配分する**
外注費・材料費・労務費などの費目ごとに、使える金額を割り振ります。

**③ 工事中に実績と比較する**
定期的に実際の発生コストと実行予算を比較し、予算超過が起きていないか確認します。

**④ 完成後に差異を分析する**
実行予算と実績の差異を記録しておくと、次の工事の見積もり精度が上がります。

## 小規模事業者でも実行予算は必要か

「うちは小さいから…」と感じる方もいると思いますが、**工事規模が小さいほど1件あたりの赤字の影響が大きくなります**。10件のうち2件が赤字なら、その2件が会社全体の利益を大きく削ります。

まずは簡易的でも、工事ごとに「請負金額・外注費・材料費・粗利」を記録する習慣から始めることで、どの工事が利益を出しているかが見えるようになります。

## まとめ

実行予算は「この工事をいくらのコストで完成させるか」の社内計画です。見積もりとは別に作ることで、工事ごとの原価管理・粗利の把握・資金繰りの見通しがしやすくなります。まずは費目ごとの上限金額を工事単位で記録することから始めてみてください。
