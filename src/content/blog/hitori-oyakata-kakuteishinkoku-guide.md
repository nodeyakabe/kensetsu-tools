## はじめに

「毎年の確定申告、どこから手をつければいいかわからない」
「どの費用が経費になるのか、いつも不安」
「工事が複数あると収支の計算がごちゃごちゃになる」

建設業の一人親方から、そんな声をよく聞きます。

確定申告は、個人事業主にとって年に一度の大仕事です。しかし、やるべきことを整理してしまえば、それほど複雑ではありません。

この記事では、建設業の一人親方を対象に、確定申告の基本から実務上のポイントまでを解説します。

---

## 1. 一人親方が確定申告をしなければならない理由

会社員であれば年末調整で税金の精算が完了しますが、一人親方（個人事業主）は自分で確定申告を行い、1年間の所得と税額を申告・納税する必要があります。

確定申告が必要な主なケース：

- 事業所得（工事収入など）がある場合
- 給与所得以外の所得が年間20万円を超える場合
- 青色申告の届出をしている場合

申告期間は毎年2月16日〜3月15日。この期間に前年（1月〜12月）の所得を申告します。

---

## 2. 青色申告と白色申告、どちらを選ぶべきか

確定申告には「青色申告」と「白色申告」の2種類があります。

### 白色申告

- 事前の届出不要
- 帳簿は収入・支出の記録のみでよい（簡易な記帳でOK）
- 特別控除なし

### 青色申告

- 事前に「青色申告承認申請書」を税務署へ提出が必要
- 複式簿記による帳簿が必要（65万円控除の場合）
- **青色申告特別控除（最大65万円）が使える**
- 赤字の3年間繰越が可能
- 家族への給与（青色事業専従者給与）を経費にできる

**一人親方には青色申告をおすすめします。**

65万円の特別控除は、課税所得をそのまま65万円減らせる大きなメリットです。たとえば所得税率20%なら、単純計算で13万円の節税効果があります。

青色申告に必要な帳簿は、Excelで管理することも可能です。複式簿記をゼロから覚えるのは大変ですが、工事ごとの収支をきちんと記録する習慣をつけることが第一歩です。

## 3. 建設業一人親方の収入と経費の考え方

### 収入（売上）

建設業の一人親方の収入は、工事ごとに元請けや発注者から受け取る請負代金です。

注意点：
- 消費税込みで受け取った場合、消費税部分は売上に含めない（免税事業者は含める場合もあり）
- 工事が年をまたぐ場合の計上時期は原則「引渡し完了日」

### 経費にできるもの（主な例）

| 費目 | 具体例 |
|------|--------|
| 材料費 | 工事に使った材料・資材の購入費 |
| 外注費 | 下請け業者や応援作業者への支払い |
| 工具・器具費 | ドリル、のこぎりなど工事用工具の購入費 |
| 車両費 | 現場への移動に使うトラック・軽トラの燃料・修繕費など |
| 通信費 | 業務用スマートフォンの通話料・データ通信費 |
| 損害保険料 | 一人親方労災保険、賠償責任保険の保険料 |
| 消耗品費 | 作業着、安全靴、ヘルメットなど |
| 接待交際費 | 発注者や協力会社との打ち合わせ飲食費（業務関連のもの） |
| 研修費 | 技術向上のための講習・セミナー費用 |

### 経費にできないもの（注意点）

- **プライベートの食事や旅行費用**（業務と無関係なもの）
- **自宅と事務所を兼ねる場合の生活費全額**（家事按分が必要）
- **家族への謝礼金**（青色事業専従者給与の届出なしのもの）

## 4. 工事台帳で収支を工事ごとに管理する

確定申告を楽にする最大のコツは、**工事ごとに収入と経費を記録しておくこと**です。

年末にまとめてレシートを整理しようとすると、「あの工事のあの材料費はいくらだったっけ？」と記憶をたどる作業になってしまいます。

工事台帳（工事ごとの収支記録）を使えば：

- 工事単位で利益が把握できる
- 決算・確定申告時の集計が一瞬で終わる
- 「この工事は赤字だった」「この現場は利益率が高い」などの分析ができる

記録しておきたい項目：

1. 工事名・発注者名
2. 請負金額
3. 工期（着工日・完工日）
4. 材料費・外注費・諸経費の内訳
5. 粗利（請負金額 − 直接費用）

Excelで工事台帳を作成して管理している一人親方も多いですが、工事数が増えると集計が面倒になりがちです。確定申告に使えるサマリーまで自動で出力できる[シンプル工事台帳 ライト版](/products/koji-daicho/)があれば、年末の作業を大幅に減らせます。

## 5. インボイス制度（適格請求書等保存方式）との関係

2023年10月から始まったインボイス制度は、一人親方の確定申告にも関係します。

### 課税事業者になった場合

適格請求書発行事業者（インボイス登録）をしている一人親方は、消費税の申告・納税も必要になります。

- 売上の消費税から、仕入れ・経費の消費税を差し引いて納税
- 請求書に登録番号・税率・税額の記載が必要
- 帳簿への記載も厳格になる

### 免税事業者のままでいる場合

- 消費税の納税は免除
- ただし、元請けが仕入税額控除できないため、取引条件の交渉が必要になるケースも
- 売上が1,000万円以下でも、状況によってはインボイス登録を検討する価値あり

どちらの選択が有利かは、取引先との関係や売上規模によって異なります。税理士への相談も選択肢のひとつです。

## 6. 確定申告の流れ（実務チェックリスト）

### 1月〜2月上旬：書類の整理

- [ ] 売上の入金記録・請求書の整理
- [ ] 経費の領収書・レシートを費目別に分類
- [ ] 外注費の支払い記録の整理
- [ ] 工事台帳の集計・最終確認

### 2月中旬〜3月上旬：申告書の作成

- [ ] 収支内訳書または青色申告決算書の作成
- [ ] 確定申告書（第一表・第二表）の作成
- [ ] e-Taxまたは税務署窓口で提出
- [ ] 消費税申告書の作成（課税事業者の場合）

### 3月15日まで：納税

- [ ] 所得税の納税（口座振替・コンビニ・e-Tax等）
- [ ] 住民税は後日、自治体から通知

## 7. まとめ：年間を通じた記録が確定申告を楽にする

建設業の一人親方が確定申告を楽にするには、**工事が終わるたびに収支を記録する習慣**が最も効果的です。

まとめると：

- **青色申告**を選んで最大65万円の控除を活用する
- **工事台帳**で工事ごとの収支を記録する
- **経費の領収書**はその場で費目を書き込んで保管する
- **インボイス対応**は売上規模と取引先の状況で判断する

年末にまとめて対応しようとすると大変ですが、日頃から記録を積み上げておけば、確定申告は「集計と提出」だけの作業になります。
